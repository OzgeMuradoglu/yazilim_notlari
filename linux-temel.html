pwd: o anda bulunan klasor yolunu gösterir.
ls: dosya içeriklerini listeler.
cd: istenilen klasorü baz alır.
/: kök dizine geri gönderir ve ordan itibaren işlem yapmak için kullanılabilir.
..: üst klasörer çıkartır.  
cd ~: ana dizine geri götürür.
sudo: yönetici olarak çalıştır.
sudo apt-get install yüklenicek programın adı: program yüklemek için.
sudo apt-get remove silinecek prograrmın adı: program silmek için.
sudo apt-get purge tamamen silinecek program adı:remove bazen yeterli olmaz purge kullanılır.
touch dosyaadı: yeni dosya oluşturma.
rm dosyaadı: dosya silme.
