1.1 Temel HTML Etiketleri

  • HTML, Hiper Metin İşaretleme Dili anlamına gelir.
  • HTML, web sayfaları oluşturmak için kullanılan standart işaretleme dilidir.
  • HTML, bir web sayfasının yapısını tanımlar.
  • HTML bir dizi öğeden oluşur.
  • HTML öğeleri, tarayıcıya içeriğin nasıl görüntüleneceğini söyler.
  • HTML öğeleri, "bu bir başlık", "bu bir paragraf", "bu bir bağlantı" vb. gibi içerik parçalarını etiketler.

Genel Yapı

  • <!DOCTYPE html> bildirim, bu belgenin bir HTML5 belgesi olduğunu tanımlar.
  • <html> öğe, bir HTML sayfasının kök öğesidir.
  • <head> öğe, HTML sayfasıyla ilgili meta bilgiler içerir.
  • <<title> öğe, HTML sayfası için bir başlık belirtir.
  • <body> öğe, HTML sayfasının gövdesini tanımlar.
  • <h1>-<h6> öğeleri, başlıklar için kullanılır.
  • <p> öğe, paragraf için kullanılır.
  • <a> öğe, link için kullanılır.
  • <img> öğe, görsel için kullanılır. Kaynak dosya ( src), alternatif metin ( alt), widthve heightöznitelikler olarak sağlanmıştır 
    <img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">

Minimal HTML Şablonu

Yeni bir HTML projesine başlarken kullanabileceğiniz en temel şablon yapısı şu şekildedir:

<!DOCTYPE html>
<html lang="tr">
<head>
    <meta charset="UTF-8">
    <title>İlk Web Sayfam</title>
</head>
<body>
    <h1>Merhaba Dünya!</h1>
    <p>Bu benim ilk web sayfam.</p>
</body>
</html>
